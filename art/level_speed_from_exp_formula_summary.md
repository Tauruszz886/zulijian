# 基于经验公式反推移动速度

- 每步经验公式：`round(1 * 等级^-0.0133 * 运动鞋倍率 * 轨迹倍率 * (1 + 好友加成%) * (1 + 游玩时间加成%) * (1 + 每日加成%) * 重生倍率)`。
- 本次按你给的测试数值折算：`运动鞋倍率=150`、`轨迹倍率=5`、`好友=25%`、`游玩时间=60%`、`每日=10%`、`重生倍率=10`，所以当前测试每步经验为 `round(16500 * 等级^-0.0133)`。
- 反推逻辑：先算 `本级升级所需移动次数 = ceil(单级经验 / 每步经验)`，再用到达当前等级前的累计移动次数映射速度。
- 速度公式：`速度 = 1 + 11 * ln(1 + 到达本级前累计移动次数) / ln(1 + 到达100级前累计移动次数)`。
- 这样 Lv1 固定为速度 1，Lv100 为速度 12；每级速度随累计成长压力上升，但用对数压缩，避免高等级经验表把速度推到不可用。
